## 기능 목록

### 페어 매칭
- [X] 순서를 랜덤으로 섞음
- [ ] 각 과정에서 두명씩 매칭
- [ ] 매칭 대상이 홀수인 경우 3인으로 구성(마지막으로 남겨서 마지막에 추가)
- [ ] 같은 레벨에서 이미 페어를 맺은 크루라면 매칭 불가
- [ ] 순서를 다시 섞어 재매칭
- [ ] (예외) 3회 시도까지 매칭이 되지 않거나 매칭 불가능이면 에러메시지 출력

### 페어 재매칭
- [ ] 문구 출력후 재매칭 여부를 입력받음
- [ ] 예를 선택할 경우 -> 페어매칭
- [ ] 아니오를 선택할 경우 코스, 레벨, 미션 재선택

### 페어 초기화
- [ ] 매칭된 모든 레벨의 페어를 초기화
